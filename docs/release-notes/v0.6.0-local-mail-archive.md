# MailMind v0.6.0 Local Mail Archive

v0.6.0 upgrades MailMind into a local email archive plus today's AI digest workspace.

## Added

- Full-history archive backfill for Gmail and IMAP mailboxes.
- `mailbox_archive_states` progress tracking.
- `email_archive_backfill` Celery jobs.
- Local archive query support for Today, Last 7 Days, Last 30 Days, Custom Range, and All Synced.
- Emails page local archive filters, archive status banner, pagination, source mailbox display, and detail metadata.
- Settings / Mailboxes Local Archive controls with confirmation before full-history sync.

## Preserved

- Dashboard daily digest semantics.
- Existing commit-then-dispatch job model.
- Mailbox-level lock behavior.

## Not Included

- Range sync jobs.
- Outlook connection.
- Historical Dashboard digest.
- Single-email AI summaries.
- Attachment download.
- Full message body sync by default.
