# v0.7.3 - Config And Diagnostics

## Scope

This release adds a desktop-only configuration and diagnostics layer to the Electron shell.

## Added

- Dedicated desktop settings window reachable from menu, tray, fallback page, and web UI
- Desktop settings route at `/settings/desktop`
- Persistent desktop config for:
  - `appUrl`
  - `healthUrl`
  - `minimizeToTray`
  - `showWindowOnStartup`
  - `notificationsEnabled`
- Main-process diagnostics snapshot and copy action
- Desktop log directory support at `<userData>/logs/desktop.log`
- Fallback page actions for:
  - retry connection
  - open desktop settings
  - open logs
  - copy diagnostics

## Changed

- Notifications now respect the desktop config flag
- Close-to-tray behavior now respects the desktop config flag
- Startup window visibility now respects the desktop config flag
- Tray and application menus now expose desktop settings and logs
