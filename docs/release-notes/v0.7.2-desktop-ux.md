# MailMind v0.7.2 Desktop UX

v0.7.2 adds the first desktop-native behavior layer on top of the Electron shell.

## Added

- System tray support for the desktop app.
- Window size, position, and maximized state persistence.
- Tray-aware close behavior on Windows and Linux.
- Desktop notifications for service disconnect and recovery events.
- Node-based desktop tests for tray policy, window state persistence, and connection transitions.

## Changed

- The desktop app menu now includes `Show MailMind`, `Hide MailMind`, and `Open Web App`.
- Closing the main window no longer exits immediately on Windows and Linux; it hides the app to the tray.

## Not Changed

- The app is still a thin Electron shell around the MailMind web UI.
- Backend, database, and worker services are still external dependencies.
- No embedded runtime, auto-update, or deep-link OAuth flow yet.

## Next

The next desktop UX pass will focus on tray menu polish, refined quit behavior, and clearer in-app diagnostics.
