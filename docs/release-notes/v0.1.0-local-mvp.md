# MailMind v0.1.0 Local MVP Release Notes

Release tag: `v0.1.0-local-mvp`

This release closes the local MVP scope for MailMind. It validates the core backend and frontend path for a local Gmail-backed AI email copilot without claiming production readiness.

## Implemented

- User registration, login, logout, and current-user lookup.
- HttpOnly cookie session creation, persistence, and revocation.
- Gmail OAuth login, callback, and disconnect.
- Mailbox connected, disconnected, active, and reauthorization state handling.
- Encrypted Gmail refresh-token storage.
- Manual Gmail "Sync Today" for messages received during the user's local day.
- Today's email list through `GET /api/emails/today`.
- Email detail through `GET /api/emails/{email_id}`.
- Gmail-backed `mark-read` and `mark-unread`.
- `user_actions` operation audit foundation.
- Backend Daily Digest generation and refresh.
- Mock AI pipeline with deterministic structured output.
- `ai_runs` records for AI execution traceability.
- `daily_digests` and `digest_items` persistence.
- `sync_jobs` persistence for sync and digest job attempts.
- Frontend theme system.
- `/settings/mailboxes` Gmail connection and sync UI.
- `/emails` email list UI.
- `/emails/[id]` email detail UI.
- Frontend mailbox sync entry point.
- Multiple frontend theme presets and light/dark mode.

## API Surface

The v0.1 backend includes:

- Auth routes under `/api/auth`.
- Gmail OAuth routes under `/api/auth/gmail`.
- Mailbox routes under `/api/mailboxes`.
- Email routes under `/api/emails`.
- Digest routes under singular `/api/digest`.
- User action routes under `/api/actions`.
- Health check at `/health`.

## Security Notes

- Do not commit `.env`.
- Do not commit Google Client Secrets.
- Do not commit `APP_ENCRYPTION_KEY`.
- Do not commit LLM API keys.
- Losing `APP_ENCRYPTION_KEY` makes saved refresh tokens undecryptable.
- Gmail restricted scopes such as `gmail.modify` require Google verification before public production use.

## Known Limitations

- Real LLM provider integration is not implemented.
- Digest frontend dashboard is not fully connected to the backend.
- Celery and background jobs are not implemented.
- Scheduled sync is not implemented.
- Multi-mailbox aggregate digest is not complete.
- Outlook and IMAP are not implemented.
- Production deployment and Google OAuth verification are not complete.
- This release is a local MVP, not a production SaaS release.
